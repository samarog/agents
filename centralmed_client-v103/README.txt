## Instalação do Agente no Website

> Este documento fornece instruções para a instalação do agente no seu site. Basta copiar e colar os snippets nos locais indicados.

1. Adicionar o Script do Agente

Na pasta fornecida, encontra-se o ficheiro [agent.js] que contém o código de execução do agente. Para o ativar, insira o código no corpo do <script> do website ou, se o ficheiro for alojado no seu servidor, crie um caminho para o mesmo no seu HTML.

<script src="*****/agent.js"></script>

2. Adicionar a Animação Personalizada

Na mesma pasta, encontra-se o ficheiro [agent-style.css], que define a animação de entrada do agente. Para aplicar o estilo, acrescente o código ao corpo <style> do website ou, se o ficheiro for alojado no seu servidor, crie um caminho para o mesmo no seu HTML.

<link rel="stylesheet" href="*****/agent-style.css">

3. Publicação e Testes

Confirme que os ficheiros [agent.js] e [agent-style.css] estão corretamente referenciados.

Carregue o site e verifique se o agente aparece e funciona corretamente.

Caso precise de assistência adicional, entre em contacto com a equipa Webtexto.

Obrigado por utilizar o nosso produto!
