## Instalação do Bot no Website

> Este documento fornece instruções para a instalação do chatbot no seu site. Basta copiar e colar os snippets nos locais indicados.

1. Adicionar o Script do Bot

Na pasta fornecida, encontra-se o ficheiro [chatbot.js] que contém o código de execução do bot. Para o ativar, insira o código no corpo do <script> do website ou, se o ficheiro for alojado no seu servidor, crie um caminho para o mesmo no seu HTML.

<script src="*****/chatbot.js"></script>

2. Adicionar a Animação Personalizada

Na mesma pasta, encontra-se o ficheiro [chatbot-style.css], que define a animação de entrada do bot. Para aplicar o estilo, acrescente o código ao corpo <style> do website ou, se o ficheiro for alojado no seu servidor, crie um caminho para o mesmo no seu HTML.

<link rel="stylesheet" href="*****/chatbot-style.css">

3. Publicação e Testes

Confirme que os ficheiros [chatbot.js] e [chatbot-style.css] estão corretamente referenciados.

Carregue o site e verifique se o bot aparece e funciona corretamente.
